Welcome to euGENia variant analisation tool!

How to start using the system:
  1. Please refer to the euGENia administrator to be registered as a user. Only authorised users can use the system.
Main sections:
  1. In the top part of the system the main sections of the euGENia system are displayed. They are Variants, Samples, Documentation and About. In the top right corner you can see your username and the corresponding list of functions. You can see your user profile there, as well as log out from the system.
  2. Variants page contains the list of main information about each genetic variant. It is also possible to filter variants
    3. and export the results into a separate file. More detailed description will be provided further.
  3. Samples page contains the list of all samples and their corresponding experiment. More detailed description will be provided further.
  4. Documentation page contains user guide to find basic information about system's functionality. More detailed description will be provided further.
  5. About page contains basic information about the euGENia system.
